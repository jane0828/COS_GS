typedef void(* gs_log_appender_console_cb_t) (gs_log_appender_t *appender, gs_log_level_t level, const gs_log_group_t *group, const gs_timestamp_t *ts, const char *format, va_list va)

Log appender for console callback type.

This callback function can be used for registering a user defined logger function if the default can not be used for the given system.

Parameters
[in]appenderpointer to the console appender.
[in]levellog level for log message
[in]grouplog group for log message
[in]tstimestamp for log message
[in]formatformat of message in printf style
[in]vavariable argument list in printf style

gs_error_t gs_log_appender_console_set_cb ( gs_log_appender_console_cb_t  cb)

Set Log appender for console callback.

When set, the given callback is called instead of the default console log function. To revert back to the default console log function, call this function with NULL as parameter.

Parameters
[in]cbcallback to use for console logging.
Returns
gs_error_t

Variable Documentation

gs_log_appender_t gs_log_appender_console

Log appender for console.

This log appender is the standard appender which is always available on any system. The appender should be registered to the root group, in order to get console/stdio logs.
