typedef void(* nanocam_img_list_cb) (int seq, cam_list_element_t *elm)

Image list callback.

This callback is called once for each element in the image listing. If no images are available, the function is called once with elm equal to NULL.

Parameters
seqElement sequence number, starting from 0.
elmPointer to image list element structure
