Download a table specification from a remote node, store it in memory and save it to local file system.

Note
Will free existing rows - do not use this on table instances with static assigned rows.

Table memory will be (re)allocated to match specification.

Parameters
[in]tinstlocal table instance.
[in]fnamename of the file to store the table specification in. If NULL, no file will be stored.
[in]nodeCSP address
[in]table_idremote table id to download.
[in]timeout_mstimeout.
[out]return_checksumfletcher16 checksum of downloaded specification "as is" - before network/host swapping.
Returns
GS_OK on success
other error code on failure
