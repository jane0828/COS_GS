typedef gs_error_t(* gs_csp_service_handler_t) (csp_conn_t *conn)

CSP Service handler.

It is the handler's responsibility to process all pakcets on the connection and close the connection when done - even if a failure is returned.

Parameters
[in]connincoming connection.
Returns
GS_OK on success
other error code on failure
