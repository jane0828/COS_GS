gs_param_table_row_t Struct Reference

Detailed Description

Table row.

A table row defines one parameter, and a table is defined by one or more rows.

Note
Make sure to update gs_param_table_checksum2(), if adding fields > 1 byte.
AVR8: Table definitions must be located in program memory, i.e. must be const.
