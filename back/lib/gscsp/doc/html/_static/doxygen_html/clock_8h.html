void gs_clock_get_time ( gs_timestamp_t *  time)

Returns real time/clock (UTC - time since Epoch/1970).

If the platform supports a Real Time Clock, the RTC is normally read on first call. An offset is calculated for the relative clock, which then is used to calculate the actual time.

Note
clock_get_time() is proto-typed in libcsp as weak, but with different argument which MUST match gs_timestamp_t.
Parameters
[out]timeuser allocated buffer, contaning the current UTC time.
