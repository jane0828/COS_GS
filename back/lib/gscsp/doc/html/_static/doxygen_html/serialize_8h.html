Enumeration Type Documentation

enum gs_param_serialize_flags_t

Serialize/deserialize flags Flags must be in range: bit 8 - 15, to avoid clash with other parts of the parameter system.

Enumerator
GS_PARAM_SF_DRY_RUN 

F_DRY_RUN do not write to memory.

GS_PARAM_SF_TO_BIG_ENDIAN 

F_TO_BIG_ENDIAN Convert from host to big endian.

GS_PARAM_SF_FROM_BIG_ENDIAN 

F_FROM_BIG_ENDIAN Confert from big endian to host order.

GS_PARAM_SF_PACKED 

F_PACKED Do not pack addresses.
