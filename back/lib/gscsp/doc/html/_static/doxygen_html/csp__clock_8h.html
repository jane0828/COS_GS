void csp_clock_get_time ( csp_timestamp_t *  time)

Get time.

This function is 'weak' in libcsp, providing a working implementation for following OS's: POSIX, Windows and Macosx. This function is expected to be equivalent to standard POSIX clock_gettime(CLOCK_REALTIME, ...).
