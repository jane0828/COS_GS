uint8_t  stop_addr,
uint16_t  timeout_ms,
int8_t *  devices 
)

Scan I2C bus for GSSB devices.

If a GSSB device is found at address, devices[address] contains 0.

Parameters
[in]start_addrI2C address to start at (included in scan).
[in]stop_addrI2C address to stop at (included in scan).
[in]timeout_msTimeout in milliseconds.
[out]devicesUser supplied array for returning results, indexed by address. Size must be at least stop_addr + 1.
