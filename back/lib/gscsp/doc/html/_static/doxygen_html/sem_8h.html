sem.h File Reference

Detailed Description

Semaphore.

The semaphore API wraps POSIX semaphore and FreeRTOS counted semaphore.

Main difference is that FreeRTOS uses different API calls, when called from within an ISR routine.

#include <gs/util/error.h>
#include <semaphore.h>
